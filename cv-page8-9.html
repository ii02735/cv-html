2019 : Outil de sondage pour les étudiants

Contexte

Au sein d'une équipe de 5, réalisation d'une application de sondage permettant aux étudiants de noter la qualité des cours.

Résultats

  • Création d'un système de rôles (admin, étudiant, intervenant)
  • Mise en place d'une architecture d'API REST via un CRUD
  • Création d'une interface web permettant aux intervenants d'ajouter des cours, et aux étudiants de les noter

Environnement

  • Docker - Docker Compose
  • HTML5 / CSS3 - Bootstrap
  • JQuery
  • NodeJS - ExpressJS
  • MongoDB
  • Webservice JSON - REST
  • VSCode
  • GitHub
  • Trello
