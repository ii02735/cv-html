2021 : Outil de suivi de statistiques sur le trafic aérien de ADP

Contexte

Au sein d'une équipe de 4, réalisation d'une application web permettant de suivre l'évolution du trafic aérien de l'Aéroport de Paris en s'appuyant sur des données de vols réalisés en 2021.

Résultats

  • Récupération de données provenant de différents fichiers CSV après nettoyage.

  • Création d'une interface web présentant des statistiques selon certains critères (départ, arrivée, météo, compagnie aérienne)

Environnement

  • Docker - Docker Compose
  • Python
  • Jupyter
  • Flask
  • MySQL
  • Webservice JSON
  • React
  • VSCode
  • GitHub
  • Trello

2020 : Outil de suivi des mots-clés sur Twitter

Contexte

Au sein d'une équipe de 4, réalisation d'une application afin de connaître la tendance au fil du temps des mots-clés / hashtags utilisés sur Twitter.

Résultats

  • Récupération périodique des derniers mots-clés utilisés, avec leurs tweets sur Twitter
  • Inscription des données récupérées dans MongoDB
  • Création d'une interface web présentant une évolution d'utilisation de chaque mot-clé en tendance sur une période donnée.

Environnement

  • Docker - Docker Compose
  • HTML5 / CSS3 - Bootstrap
  • JQuery
  • Nginx
  • API Twitter
  • NodeJS - ExpressJS
  • MongoDB
  • VSCode
  • GitHub
  • Trello
